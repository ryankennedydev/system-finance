# Controle Financeiro com Python e SQLite

Sistema simples de controle financeiro feito em Python, utilizando SQLite como banco de dados.

## Português

### Sobre o projeto

Este projeto permite cadastrar, listar, editar e remover transações financeiras pelo terminal.

O objetivo é praticar conceitos de Python, SQLite, funções, CRUD e manipulação de banco de dados.

### Funcionalidades

* Adicionar transações
* Editar transações
* Listar todas as transações
* Buscar transação por ID
* Remover transações
* Salvar os dados em um banco SQLite
* Validar o tipo da transação como receita ou despesa

### Tecnologias utilizadas

* Python
* SQLite

## English

### About the project

This project is a simple financial control system built with Python and SQLite.

It allows users to add, list, edit, and remove financial transactions through the terminal.

The goal of this project is to practice Python, SQLite, functions, CRUD operations, and database management.

### Features

* Add transactions
* Edit transactions
* List all transactions
* Search transaction by ID
* Remove transactions
* Store data in a SQLite database
* Validate the transaction type as income or expense

### Technologies used

* Python
* SQLite
